What is Regression Testing?

  • Worrying that refactoring code will break your app?
  • Worrying that major code changes will break your app?
  • Don't worry
- Uncover software bugs after enchanchments, bug fixes, etc. - Make sure that changes doesn't introduce new issues

PHPUnit

			public function testDistance() {
			  $coord = new Coordinate(5, 0);
			  $line = new Line(
			    new Coordinate(0, 0),
			    new Coordinate(5, 5)
			  );
			  $dist = round($coord->distanceToLine($line), 2);
			  $this->assertEquals(3.54, $dist);
			}
- Good for testing business logic - Test individual units of code - Given input X verify that output Y is returned

Selenium & Selenium WebDriver

  • Testing framework for web applications
  • Hooks into browser
    • Works with most platforms using JavaScript to perform actions
  • Exposes JsonWireProtocol interface (REST API)
  • Clients for PHP, Ruby, Java, etc.
- Allows us to send commands to the browser and read values

Getting ready

  • PHPUnit
  • Selenium Server
    • Chrome WebDriver
    • java -jar /usr/bin/selenium-server-standalone-2.44.0.jar
  • composer.json: phpunit/phpunit-selenium
    • composer update

Controlling Selenium from PHP(Unit)

  • Finding
    • $this->byClassName(...)
    • $this->byCssSelector(...)
    • $this->byId(...)
    • $this->byXPath(...)
  • Reading
    • $element->text()
    • $element->css('font-size')
    • $element->displayed()
    • $element->attribute('alt')
  • Interacting
    • $element->click()
    • $element->clear()
  • Typing
    • $this->keys(...)
